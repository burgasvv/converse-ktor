
# ___Converse-Messenger API___

## Версии
+ Ktor 3.4.0
+ OpenJDK 25
+ Gradle 9.2.1

## Основной стек зависимостей
+ Exposed DB
+ Postgresql
+ Security Basic
+ Routing
+ Serialization
+ Redis
+ Kafka with Raft
+ Websocket

## Описание
    Server API messenger для работы с контактами, диалогами, чатами, сообществами, отправки публикаций и сообщений
